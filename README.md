# fsglib

Fine Star Guiding Library for ET algorithm verification.

## Overview

This repository contains the Python package, configuration, examples, scripts, and tests used for fine star guiding related algorithm development and verification.

## Structure

- `fsglib/`: package source code
- `configs/`: configuration files
- `docs/`: project notes and documentation
- `examples/`: example usage
- `scripts/`: utility and workflow scripts
- `tests/`: test suite

## Development

Install dependencies:

```bash
pip install -r requirements.txt
```

Run tests:

```bash
pytest
```
